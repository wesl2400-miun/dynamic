# Frontend-baserad webbutveckling
## Laboration 5
Uppgigften går ut på att lägga till två nya sidor till den redan existerande webbplatsen från föregående laborationer. Den första sida ska heta diagram.html och innehålla stapel- och cirkeldiagram för mest sökta kurser och program som hämtas från en URL.

Den andra sidan ska heta karta.html och innehålla ett sökformulär för en plats. Platsnamnet ska skickas till ett API och returnera koordinater som används sedan för att bygga en karta. För övrigt ska alla JS-filer dokumenteras med JSDoc och den genererade dokumentationen publiceras online.

### Länken till JSDoc-dokumentationen -> https://wesl2400-miun.github.io/dynamic-docs/
